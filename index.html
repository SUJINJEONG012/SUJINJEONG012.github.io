03

What I can do

주도적인 마음과 창의적인 생각을 가진 사람이 가장 강력한 인재라고 생각합니다.
경청할 자세와 성실함을 기본적으로 가지고 있으며, 주도적으로 자기계발을 하고 있습니다.

  • 자바

    • - 객체지향 프로그래밍에 대해 이해
    • - Java에서 제공하는 클래스와 메소드를 적절하게 사용
    • - 자바의 기초를 공부하고 자료구조와 알고리즘을 자바기반으로 학습

  • Spring & SpringBoot

    • - Legacy Project를 이용하여 웹 애플리케이션 구현 경험
    • - Mybatis 를 사용하여 웹개발을 구현 경험
    • - (With thymeleaf or JSP) : 사이드 프로젝트로 애플리케이션 구현 경험
    • - Spring으로 REST API를 활용하여 웹 서비스를 개발
    • 스프링 마이바티스를 이용하여 세션값이용과 스프링 Security Oauth2를 사용해서 카카오 로그인 구현
    • - JPA : 사이드 프로젝트로 애플리케이션 구현 경험
    • - jQuery : Vanila JS와 같이 혼용하여 사용 경험

  • MySQL

    • - MySQL의 기본 쿼리문을 활용하여 데이터베이스 사용 경험
    • - 테이블 CRUD 및 조인 등 프로젝트를 통해 사용 경험 있습니다.

  • 형상관리 툴

    협업효율성과 충돌관리와 백업 및 복구를 위해 사용

    • - Git : CLI 에서 commit 및 원격저장소(GitHub로 push)
    • - Gitkraken : GUI 사용 경험이 있습니다.
    • - SVN : 코드의 버전 관리 및 협업을 위해 SVN을 사용하여 프로젝트를 진행

  • 리눅스

    • - 서버 재기동 및 디스크 용량 관리를 위해 시스템의 안정성을 유지
    • - 서버다운타임을 최소화
    • - 불필요한 파일삭제를 통해서 서버자원을 효율적으로 관리

  • OAuth2.0

    • - 카카오 로그인 API 연동
    • - 사용자 인증 과정을처리하고, 카카오서버로부터 액세스토큰을 받아 사용자의 정보를 가져오는 과정을 구현

  • AWS

    • - 아마존 AWS를 활용해서 웹 서비스를 배포한 경험이 있습니다.
    • - EC2 가상 ,RDS(MySQLDB)를 사용해봤습니다.
